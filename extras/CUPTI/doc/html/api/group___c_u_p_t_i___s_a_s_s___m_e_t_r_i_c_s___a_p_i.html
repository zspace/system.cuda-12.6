5.13.2. EnumerationsÔÉÅ

enum CUpti_SassMetrics_OutputGranularityÔÉÅ

Values:

enumerator CUPTI_SASS_METRICS_OUTPUT_GRANULARITY_GPUÔÉÅ

SASS metric data will be collected at GPU level. In CUpti_SassMetricsGetDataProperties_Params struct the numOfInstances will be equal to 1.

enumerator CUPTI_SASS_METRICS_OUTPUT_GRANULARITY_SMÔÉÅ

SASS metric data will be collected at SM level In CUpti_SassMetricsGetDataProperties_Params struct the numOfInstances will be equal to number of SMs in the GPU.

enumerator CUPTI_SASS_METRICS_OUTPUT_GRANULARITY_SMSPÔÉÅ

SASS metric data will be collected at SM sub-partition level In CUpti_SassMetricsGetDataProperties_Params struct the numOfInstances will be equal to number of SM sub-partitions in the GPU.

enumerator CUPTI_SASS_METRICS_OUTPUT_GRANULARITY_INVALIDÔÉÅ

CUptiResult cuptiSassMetricsDisable(CUpti_SassMetricsDisable_Params *pParams)ÔÉÅ

SASS metric data collection disable API will mark the end of a range, any kernel launched after this API call will not be profiled for the SASS metrics.

Parameters

pParams ‚Äì A pointer to CUpti_SassMetricsDisable_Params

Return values

  • CUPTI_SUCCESS ‚Äì

  • CUPTI_ERROR_INVALID_PARAMETER ‚Äì if any pParams is not valid

  • CUPTI_ERROR_NOT_SUPPORTED ‚Äì indicates that the system/device doesn‚Äôt support SASS metric data collection

  • CUPTI_ERROR_INVALID_CONTEXT ‚Äì if any cuda context has not been created prior to this API call

  • CUPTI_ERROR_INVALID_OPERATION ‚Äì if this API is called multiple times for a cuda context without calling cuptiSassMetricsEnable() API or called before cuptiSassMetricsSetConfig() API call.

CUptiResult cuptiSassMetricsFlushData(CUpti_SassMetricsFlushData_Params *pParams)ÔÉÅ

Flush SASS metrics data from CUPTI internal buffer to the user buffer.

User needs to allocate the buffer for retrieving the data. The number of records collected can be queried using the API cuptiSassMetricsGetDataProperties().

Parameters

pParams ‚Äì A pointer to CUpti_SassMetricsFlushData_Params

Return values

  • CUPTI_SUCCESS ‚Äì

  • CUPTI_ERROR_INVALID_PARAMETER ‚Äì if any pParams is not valid

  • CUPTI_ERROR_NOT_SUPPORTED ‚Äì indicates that the system/device doesn‚Äôt support SASS metric data collection.

  • CUPTI_ERROR_INVALID_OPERATION ‚Äì if this API is called outside the enable/disable range.

CUptiResult cuptiSassMetricsGetDataProperties(CUpti_SassMetricsGetDataProperties_Params *pParams)ÔÉÅ

SASS metric data properties API will give the data regarding number of instances of a metric value and number of SASS instruction data has been collected.

The number of instances of a metric will vary as per user set the output granularity level with CUpti_SassMetrics_OutputGranularity value. User need to allocate memory for retriving the SASS data using cuptiSassMetricsFlushData() API.

Parameters

pParams ‚Äì A pointer to CUpti_SassMetricsGetDataProperties_Params

Return values

  • CUPTI_SUCCESS ‚Äì

  • CUPTI_ERROR_INVALID_PARAMETER ‚Äì if any pParams is not valid

  • CUPTI_ERROR_NOT_SUPPORTED ‚Äì indicates that the system/device doesn‚Äôt support SASS metric data collection

  • CUPTI_ERROR_INVALID_OPERATION ‚Äì if this API is called outside the enable/disable range.

CUptiResult cuptiSassMetricsSetConfig(CUpti_SassMetricsSetConfig_Params *pParams)ÔÉÅ

Set config for the SASS metric data collection for a device.

User need to call this API before calling any of the SASS metric data collection APIs. Each set config API call need to be followed by cuptiSassPatchingUnSetConfig API before calling the cuptiSassMetricsSetConfig() API again for the same device.

Parameters

pParams ‚Äì A pointer to CUpti_SassMetricsSetConfig_Params

Return values

  • CUPTI_SUCCESS ‚Äì

  • CUPTI_ERROR_INVALID_PARAMETER ‚Äì if any pParams is not valid

  • CUPTI_ERROR_INVALID_CONTEXT ‚Äì if any cuda context has not been created prior to this API call

  • CUPTI_ERROR_INVALID_OPERATION ‚Äì if this is called multiple times for the device without calling unset config API

  • CUPTI_ERROR_NOT_SUPPORTED ‚Äì indicates that the system/device doesn‚Äôt support SASS metric data collection

CUptiResult cuptiSassMetricsUnsetConfig(CUpti_SassMetricsUnsetConfig_Params *pParams)ÔÉÅ

Unset config API will reset the SASS metric data collection configuration for the device.

Once this API called CUPTI will deallocate all the memory allocated and remove all the configuration for SASS metric data collection. User can only call this API for a device where cuptiSassMetricsSetConfig() API has been called earlier for the device.

Parameters

pParams ‚Äì A pointer to CUpti_SassMetricsSetConfig_Params

Return values

  • CUPTI_SUCCESS ‚Äì

  • CUPTI_ERROR_INVALID_PARAMETER ‚Äì if any pParams is not valid

  • CUPTI_ERROR_INVALID_CONTEXT ‚Äì if any cuda context has not been created prior to this API call

  • CUPTI_ERROR_INVALID_OPERATION ‚Äì if this is called multiple times for the device without calling set config API

  • CUPTI_ERROR_NOT_SUPPORTED ‚Äì indicates that the system/device doesn‚Äôt support SASS metric data collection
